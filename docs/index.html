Web application designed to streamline the task management of a particular group of people, or even companies.

Login and Register

A user can login and register in our website. A not logged in user can access public projects.

Project Page

Projects have coordinators and team members. The coordinators can create tasks and group them in sprints ,giving them a deadline. These tasks can be assigned, claimed and completed by team members, as well as coordinators. Besides this, it is possible to comment on each task, and enter the task page, which contains more information about its goal.

Every projects also has a forum, where the team members can discuss several topics. Adding to this, it is also possible to consult the project statistics and the member lists.

The project coordinators have more options besides the creation and changes of tasks and sprints. These options consist of handling requests and invites to the project, the promotion and expulsion of members.

Profile Page

Inside the profile page, the user can check it's projects, remove itself from it, check their statistics and edit their profile.
